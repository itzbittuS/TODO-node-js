A clean, fast, and elegant **Todo CRUD application** built with **Node.js, Express, and EJS** featuring a stunning dark UI with glassmorphism effects.


---

### Features

- Add new tasks with title & description  
- Edit tasks (title, description, status, category)  
- Mark tasks as **Pending** or **Completed**  
- Delete tasks with confirmation  
- Fully responsive & mobile-friendly  
- Beautiful dark theme with glassmorphism cards  
- Smooth hover effects and modern UI  
- Zero dependencies beyond Express & EJS

### Tech Stack

- **Backend: `Node.js` + `Express`  
- View Engine: `EJS`  
- Styling: Pure `CSS` (Glassmorphism + Dark Mode)  
- Data Storage: In-memory array (easy to upgrade to MongoDB later)  
- No database required (perfect for learning or quick projects)
